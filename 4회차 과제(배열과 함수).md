> 과제 1. 사용자의 입력으로 배열 다루기
 - 4개의 정수를 담을 수 있는 배열을 생성, 사용자에게 값을 입력받고 내용을 출력하는 프로그램을 작성하시오
    - int 4개를 담을 수 있는 배열 선언
    - "1번 요소를 입력하여 주십시오" 출력 후 사용자 입력 받기
    - 나머지 요소들도 동일한 방법으로 입력 받기
    - 모든 요소에 대한 입력이 끝났다면 "입력된 요소는 다음과 같습니다" 라는 텍스트 이후 배열의 모든 데이터를 출력
  
### 답)
```csharp
 static void Main(string[] args)
 {
     int[] numbers = new int[4];
     Console.WriteLine("4개의 정수를 입력하여 주십시오!");
     for (int i = 0; i < numbers.Length; i++)
     {
         Console.Write($"{i + 1}번 요소: ");
         numbers[i] = int.Parse(Console.ReadLine());
     }
     Console.Write("\n입력된 요소는 다음과 같습니다:");

     //Console.WriteLine(string.Join(", ", numbers));
     string result = "";
     bool isFirst = true;

     foreach (int num in numbers)
     {
         if (!isFirst)
         {
             result += ", ";
         }
         result += num;
         isFirst = false; // 한 번 실행된 후에는 false로 변경
     }

     Console.WriteLine(result);
 }

```
  

> 과제 2. 가장 큰 수 구하기
 - 4개의 정수를 인자로 받아 가장 큰 수를 출력하는 함수를 제작하시오.

### 답)
```csharp
static int Max(int a, int b, int c, int d)
{
    int max = a;
    if (b > max)
        max = b;
    if (c > max)
        max = c;
    if (d > max)
        max = d;
    return max;
}
```

> 과제 3. 다차원 배열과 함수 다루기
 - int 타입의 2차원 배열에 데이터를 추가하고, 데이터를 교환하는 프로그램을 작성하시오
    - int[4, 4] 크기의 배열 선언
    - 반복문을 통해 순서대로 3의 배수들로 채워넣기
    - int타입의 인자를 네 개(a, b, c, d) 입력받아 배열 내에 있는 [a, b]데이터와 [c, d] 데이터를 서로 교체하는 함수를 제작
    - [4, 4]의 형태로 들어있는 정수들을 출력

### 답)
```csharp
using System;
using System.Diagnostics;

namespace Array
{
    internal class Program
    {
        static void Main(string[] args)
        {
            Console.WriteLine(Max(3, 6, 9, 12));

            //  int 타입의 2차원 배열에 데이터를 추가하고, 데이터를 교환하는 프로그램을 작성하시오
            //  int[4, 4]  크기의 배열 선언
            //  반복문을 통해 순서대로 3의 배수들로 채워넣기
            //  int 타입의 인자를 네 개(a, b, c, d) 입력받아 배열 내에 있는 [a, b]데이터와 [c, d] 데이터를 서로 교체하는 함수를 제작
            //  [4, 4] 의 형태로 들어있는 정수들을 출력

            int[,] Matrix = new int[4, 4];
            int value = 3;

            for (int i = 0; i < 4; i++)
            {
                for (int j = 0; j < 4; j++)
                {
                    Matrix[i, j] = value;
                    value += 3;
                }
            }

            Console.WriteLine("--- 교환 전 배열 상태 ---");
            PrintMatrix(Matrix);
            SwapNumbers(Matrix, 0, 0, 3, 3);
            Console.WriteLine("--- 교환 후 배열 상태 ---");
            PrintMatrix(Matrix);

        }

        static void SwapNumbers(int[,] array, int a, int b, int c, int d)
        {
            int temp = array[a, b];
            array[a, b] = array[c, d];
            array[c, d] = temp;
        }

        static void PrintMatrix(int[,] matrix)
        {
            for (int i = 0; i < matrix.GetLength(0); i++)
            {
                for (int j = 0; j < matrix.GetLength(1); j++)
                {
                    Console.Write(matrix[i, j] + "\t");
                }
                Console.WriteLine();
            }
        }
    }
}
```

> 과제 4(심화/선택사항). 피보나치 함수 제작

- 피보나치 수란, 맨 첫째와 둘째 항은 각각 1이지만, 그 뒤로 오는 모든 항은 앞의 두 항의 합인 수열입니다. 예를 들어 [1,1,2,3,5,8,...] 이렇게 4번째(3) 항목은 2번째(1)와 3번째(2)의 합이고, 마찬가지로 6번째(8) 항목은 바로 앞 4번째(3), 5번째(5)의 합인 형태입니다. 함수의 인자값으로 '몇번째' 인지 정수로 건네주면, 일치하는 피보나치 수를 반환하는 함수를 제작하세요. 참고로 11번째 피보나치 수는 89, 20번째 피보나치 수는 6765 입니다.
- 해당 과제를 수행할 경우 제출용 마크다운 문서에 내용을 추가하여 제출합니다

    ※ 작동은 하는데, 특정 수에서 터진다면 해당 코드도 킵해두시고, 안터지는 다른 방법도 고민해보시기 바랍니다

### 답)
```csharp
 static int Fibonacci(int n)
 {
     if (n <= 1)
         return n;

     return Fibonacci(n - 1) + Fibonacci(n - 2);
 }
```
