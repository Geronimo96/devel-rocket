> 과제 2. 간단한 계산 프로그램 제작
- 십의 자리의 정수를 입력받아 십의 자리의 숫자와 일의 자리의 숫자를 연산, 출력하는 프로그램을 작성하시오.
    - 첫 줄의 입력은 한 줄로 처리하며, 십의 자리의 숫자를 a, 일의 자리의 숫자를 b로 변수를 선언해 관리한다.
    - 첫 번째 출력 항목 : 두 결과를 더한 값
    - 두 번째 출력 항목 : 첫번째 값에서 두번째 값을 뺀 값
    - 세 번째 출력 항목 : 두 결과를 곱한 값
    - 네 번째 출력 항목 : 첫번째 값에서 두번째 값을 나눈 값
    - 다섯번째 출력 항목 : 첫번째 값에서 두번째 값을 나눈 나머지

```csharp
using System;

namespace _251209_Hello_World
{
    internal class Program
    {
        static void Main(string[] args)
        {   // 블록범위 시작
            bool isCorrect; // 정수인지 판별

            // 1. 입력 처리
    
            Console.Write("십의 자리 정수를 입력하세요 (예: 47): ");
            isCorrect = int.TryParse(Console.ReadLine(), out int inputNumber) && inputNumber >= 10 && inputNumber <= 99;

            Console.WriteLine($"제대로된 숫자가 입력이 되었나요? {isCorrect}");

            // 2. 변수 선언 및 자릿수 분리
            // 십의 자리 숫자 (a): 정수 나눗셈 (/)을 이용하여 10으로 나눈 몫을 구합니다.
            int a = inputNumber / 10;

            // 일의 자리 숫자 (b): 나머지 연산자 (%)를 이용하여 10으로 나눈 나머지를 구합니다.
            int b = inputNumber % 10;

            // 출력 항목
            Console.WriteLine($"\n입력된 숫자: {inputNumber}");
            Console.WriteLine($"십의 자리 숫자 (a): {a}");
            Console.WriteLine($"일의 자리 숫자 (b): {b}");
            Console.WriteLine("----------------------------------");

            // 3. 연산 및 출력

            // 첫 번째 출력 항목: 두 결과를 더한 값 (a + b)
            int sum = a + b;
            Console.WriteLine($"1. 두 숫자를 더한 값 (a + b): {sum}");

            // 두 번째 출력 항목: 첫 번째 값에서 두 번째 값을 뺀 값 (a - b)
            int difference = a - b;
            Console.WriteLine($"2. 두 숫자를 뺀 값 (a - b): {difference}");

            // 세 번째 출력 항목: 두 결과를 곱한 값 (a * b)
            int product = a * b;
            Console.WriteLine($"3. 두 숫자를 곱한 값 (a * b): {product}");

            // 네 번째 출력 항목: 첫 번째 값에서 두 번째 값을 나눈 값 (a / b)
            // (주의: 정수 나눗셈 결과이므로 소수점 이하는 버려집니다.)
            int quotient = a / b;
            Console.WriteLine($"4. 두 숫자를 나눈 값 (a / b): {quotient}");

            // 다섯 번째 출력 항목: 첫 번째 값에서 두 번째 값을 나눈 나머지 (a % b)
            int remainder = a % b;
            Console.WriteLine($"5. 두 숫자를 나눈 나머지 (a % b): {remainder}");
        }   // 블록범위 끝
    }
}
```
> 과제 3(심화/선택사항). 실수의 부정확함
 - 아래 코드에서 True가 출력되지 않는 이유를 마크다운 문서에 서술하시오

```csharp
double a = 0.1 + 0.2;
double b = 0.3;

Console.WriteLine(a == b);  // False 출력
```
답)
```
--------0.1 + 0.2 ≠ 0.3인 이유--------
0.1₁₀ (저장값) ≈ 0.1000000014901161...
0.2₁₀ (저장값) ≈ 0.2000000029802322...
합계           ≈ 0.3000000044703484...

0.3₁₀ (저장값) ≈ 0.2999999940395355...

결과: 0.1 + 0.2 ≠ 0.3 (미세한 오차 존재)

--------정확히 변환되는 경우는 문제없음--------
0.25 + 0.5 = 0.75 (정확한 계산)
이유: 0.25₁₀ = 0.01₂, 0.5₁₀ = 0.1₂, 0.75₁₀ = 0.11₂
모두 이진법으로 정확히 표현 가능
```
